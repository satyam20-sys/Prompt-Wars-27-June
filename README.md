# Prompt-Wars-27-June
Mental Wellness Tracker
Description

The AI-Powered Mental Wellness Tracker is a web application designed to help users monitor and improve their mental well-being through intelligent, personalized guidance. Users can record their daily mood, stress levels, sleep quality, and journal their thoughts using a simple and intuitive interface.

The application leverages the Gemini 3.5 Flash API to analyze user inputs and generate personalized wellness insights, coping strategies, motivational messages, and self-care recommendations. By identifying emotional patterns and providing actionable suggestions, the system encourages users to develop healthy habits, improve emotional awareness, and maintain consistent mental wellness.

Built with HTML, CSS, and Vanilla JavaScript for the frontend and Flask (Python) for the backend, the application offers a lightweight, responsive, and scalable solution while ensuring a smooth user experience.

Features
1.Daily mood and emotion tracking
2.Journal entry logging
3.Stress level and sleep quality monitoring
4.AI-powered wellness insights
5.Personalized coping strategies
6.Motivational messages
7.Responsive and user-friendly interface
8.Secure API integration with Gemini 3.5 Flash
